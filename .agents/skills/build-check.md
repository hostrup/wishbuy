name: build-check
description: Validerer at SvelteKit-applikationen bygger fejlfrit
---

Din opgave er at bekræfte, at applikationen kan bygge til produktion uden fejl.

### Arbejdsproces:
1. Kør `npm run build` i projektroden.
2. Tjek outputtet for Svelte compiler fejl eller TypeScript issues.
3. Rapporter eventuelle fejl til mig.
